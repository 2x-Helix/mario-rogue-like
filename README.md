# Mario Rogue Like Assignment
## Monash FIT2099 Object oriented design and implementation S1-2022
This project is a console-based 2D rouge-like game completed as the semester assignment of FIT2099 (2022-Sem1).
This was done in collaboration with [ChunKau Mok](https://github.com/petermok-1213), [James Huynh](https://github.com/James-JH) and [Matthew Siegenthaler](https://github.com/2x-Helix/).
